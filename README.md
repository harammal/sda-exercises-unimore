# Strutture Dati e Algoritmi Exercises

This repository contains my personal solutions to the exercises of the course "Strutture Dati e Algoritmi" (Data Structures and Algorithms) at the University of Modena and Reggio Emilia during the academic year 2024/2025.

These solutions are shared for educational purposes only.

## Repository Structure

- `01-recursion/` – Recursive exercises
- `02-backtracking/` – Backtracking exercises
- `03-greedy/` – Greedy algorithm exercises
- `04-sorting/` – Sorting exercises
  - `primitives/`
  - `exercises/`
- `05-linked-lists/` – Linked list exercises
  - `primitives/` 
  - `exercises/`
- `06-trees-and-bst/` – Tree and BST exercises
  - `primitives/`
  - `exercises/`
- `07-heaps/` – Heap exercises
  - `primitives/`
  - `exercises/`
