# Strutture Dati e Algoritmi Exercises

This repository contains my personal solutions to the exercises of the course "Strutture Dati e Algoritmi" (Data Structures and Algorithms) at the University of Modena and Reggio Emilia during the academic year 2024/2025.

These solutions are shared for educational purposes only.
